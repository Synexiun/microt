# CMS Admin Panel Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate services and site constants from hardcoded TypeScript into Vercel Blob JSON, and build admin UI pages so the owner can edit content without code changes.

**Architecture:** Two new Vercel Blob documents (`data/services.json`, `data/site-content.json`) act as the data store, seeded from existing static defaults on first read. Five new API routes under `/api/admin/` handle reads and writes. Three new admin pages expose forms for editing. Client components that cannot call async functions keep the static defaults.

**Tech Stack:** Next.js 14 App Router, Vercel Blob, Node.js crypto (inline auth), Tailwind CSS, existing UI components (Card, Input, Button, Spinner).

### Task 1: Add `readJsonObject`/`writeJsonObject` to `data.ts`

**Files:**
- Modify: `src/lib/data.ts`

- [ ] **Step 1:** Add two helpers at the bottom of `src/lib/data.ts`.
  Pattern: same as existing `readJsonFile`/`writeJsonFile` but without array wrapping.
  Both use the `list`/`put` imports already at the top of the file.

  ```ts
  export async function readJsonObject<T>(filename: string): Promise<T | null> {
    try {
      const { blobs } = await list({ prefix: `data/${filename}` });
      if (blobs.length === 0) return null;
      const token = process.env.BLOB_READ_WRITE_TOKEN;
      const response = await fetch(blobs[0].url, {
        headers: token ? { authorization: `Bearer ${token}` } : {},
      });
      if (!response.ok) return null;
      return (await response.json()) as T;
    } catch {
      return null;
    }
  }

  export async function writeJsonObject<T>(filename: string, data: T): Promise<void> {
    await put(`data/${filename}`, JSON.stringify(data), {
      access: "private",
      addRandomSuffix: false,
    });
  }
  ```

### Task 9: `PUT /api/admin/services/[slug]`

**Files:**
- Create: `src/app/api/admin/services/[slug]/route.ts`

**Pattern:** Read all services → find by slug → merge updates (excluding slug field) → write.
(`updateInJsonFile` requires `id: string`, incompatible with slug-based Service identity.)

- [ ] **Step 1:** Create the route:

  ```ts
  import { NextRequest, NextResponse } from "next/server";
  import { cookies } from "next/headers";
  import { validateSession, SESSION_COOKIE_NAME } from "@/lib/auth";
  import { getServices } from "@/lib/services";
  import { writeJsonFile } from "@/lib/data";
  import type { Service } from "@/types";

  export const dynamic = "force-dynamic";

  export async function PUT(
    request: NextRequest,
    { params }: { params: Promise<{ slug: string }> }
  ) {
    const cookieStore = await cookies();
    const token = cookieStore.get(SESSION_COOKIE_NAME)?.value;
    if (!token || !validateSession(token)) {
      return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
    }
    const { slug } = await params;
    try {
      const body = (await request.json()) as Partial<Service>;
      const services = await getServices();
      const index = services.findIndex((s) => s.slug === slug);
      if (index === -1) return NextResponse.json({ error: "Not found" }, { status: 404 });
      const { slug: _s, ...updates } = body;
      void _s;
      services[index] = { ...services[index], ...updates };
      await writeJsonFile("services.json", services);
      return NextResponse.json(services[index]);
    } catch {
      return NextResponse.json({ error: "Failed" }, { status: 500 });
    }
  }
  ```

### Task 10: `PUT /api/admin/services/[slug]/image`

**Files:**
- Create: `src/app/api/admin/services/[slug]/image/route.ts`

**Logic:** If `service.image` starts with `https://`, call `del()` on it before uploading the new file.
Local paths (starting with `/`) are not deleted — they live in the repo, not Blob storage.

- [ ] **Step 1:** Create the route:

  ```ts
  import { NextRequest, NextResponse } from "next/server";
  import { cookies } from "next/headers";
  import { validateSession, SESSION_COOKIE_NAME } from "@/lib/auth";
  import { getServices } from "@/lib/services";
  import { writeJsonFile } from "@/lib/data";
  import { put, del } from "@vercel/blob";

  export const dynamic = "force-dynamic";

  export async function PUT(
    request: NextRequest,
    { params }: { params: Promise<{ slug: string }> }
  ) {
    const cookieStore = await cookies();
    const token = cookieStore.get(SESSION_COOKIE_NAME)?.value;
    if (!token || !validateSession(token)) {
      return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
    }
    const { slug } = await params;
    try {
      const formData = await request.formData();
      const file = formData.get("file") as File | null;
      if (!file) return NextResponse.json({ error: "No file" }, { status: 400 });
      const services = await getServices();
      const index = services.findIndex((s) => s.slug === slug);
      if (index === -1) return NextResponse.json({ error: "Not found" }, { status: 404 });
      const oldImage = services[index].image;
      if (oldImage.startsWith("https://")) {
        try { await del(oldImage); } catch { /* ignore */ }
      }
      const ext = file.name.split(".").pop() || "jpg";
      const blob = await put(`services/${slug}.${ext}`, file, {
        access: "public", addRandomSuffix: false,
      });
      services[index] = { ...services[index], image: blob.url };
      await writeJsonFile("services.json", services);
      return NextResponse.json({ url: blob.url });
    } catch {
      return NextResponse.json({ error: "Failed" }, { status: 500 });
    }
  }
  ```

### Task 11: `GET`/`PUT /api/admin/content`

**Files:**
- Create: `src/app/api/admin/content/route.ts`

**PUT merge logic:** Each top-level key present in the body overrides the corresponding
stored value (shallow merge per key: `brand`, `businessHours`, `socialLinks`).

- [ ] **Step 1:** Create the route:

  ```ts
  import { NextRequest, NextResponse } from "next/server";
  import { cookies } from "next/headers";
  import { validateSession, SESSION_COOKIE_NAME } from "@/lib/auth";
  import { getSiteContent } from "@/lib/constants";
  import { writeJsonObject } from "@/lib/data";
  import type { SiteContent } from "@/lib/constants";

  export const dynamic = "force-dynamic";

  async function authorize(): Promise<boolean> {
    const cookieStore = await cookies();
    const token = cookieStore.get(SESSION_COOKIE_NAME)?.value;
    return !!(token && validateSession(token));
  }

  export async function GET() {
    if (!(await authorize())) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
    try {
      return NextResponse.json(await getSiteContent());
    } catch {
      return NextResponse.json({ error: "Failed" }, { status: 500 });
    }
  }

  export async function PUT(request: NextRequest) {
    if (!(await authorize())) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
    try {
      const current = await getSiteContent();
      const body = (await request.json()) as Partial<SiteContent>;
      const updated: SiteContent = {
        brand: body.brand ? { ...current.brand, ...body.brand } : current.brand,
        businessHours: body.businessHours ?? current.businessHours,
        socialLinks: body.socialLinks ? { ...current.socialLinks, ...body.socialLinks } : current.socialLinks,
      };
      await writeJsonObject("site-content.json", updated);
      return NextResponse.json(updated);
    } catch {
      return NextResponse.json({ error: "Failed" }, { status: 500 });
    }
  }
  ```
